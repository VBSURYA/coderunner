## ✨ Features

- **Zero Configuration:** Start coding instantly without setting up local runtime environments.
- **Multi-Language Support:** Write and compile scripts in popular programming languages (Python, JavaScript, TypeScript, C++, Java, HTML/CSS preview, and more).
- **Interactive Terminal:** Execution console featuring standard output, error streams, and real-time execution telemetry.
- **File System Tree:** Create, rename, delete, and switch between project files easily.
- **Git Panel:** Simulated version control integrations to track changes, review diffs, and inspect working directories.
- **Ad-Supported Ecosystem:** Non-intrusive monetization with built-in native banners and popunder support.
- **Privacy First:** Built-in cookie-consent policies and sandbox environment compliance.

## 💻 Running Locally

Follow these simple steps to run CodeRunner on your local machine:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed.

### 2. Clone & Setup
```bash
# Clone the repository and navigate to the directory
cd coderunner

# Install project dependencies
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
Your local server will spin up, typically at `http://localhost:5173`.
